## Project describtion:
This project simulates the embedded controller in a microwave oven, with all the drivers bulit from scratch ( LCD, Keypad, GPIO, Timers ).
 
## Components:
1. LCD 1602a.
2. Keypad 4x4.
3. Tiva-c tmc4h123gh6pm microcontroller.
4. External push buttons.
